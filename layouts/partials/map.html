Trains in Georgia

Hello there! This is a train map of Georgia. Here you can see the main train routes, find out their schedules and which cities they pass through.

The site is under construction, but all the main routes have already been added.

Last update: 01.01.2025

About

This map has been created to help you navigate Georgia's railway system. Use it in conjunction with other sources of information for a more complete picture.

The official website of Georgian Railways: https://www.railway.ge/en/traffic-general-schedule/.

An amazing guide to all aspects of rail travel in Georgia by Emily Lush: https://wander-lush.org/trains-in-georgia-country-tips-map/.
